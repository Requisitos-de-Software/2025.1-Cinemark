# Verificação do grupo

## Introdução

Este documento apresenta a **lista de verificação** (versão 2.0) elaborada para avaliar os artefatos produzidos na Entrega 3: *Modelagem: Casos de Uso, Especificação Suplementar, Cenários e Léxicos*. As tabelas foram elaboradas seguindo o modelo de lista de verificação sugerida pelo professor, [André Barros de Sales](https://sigaa.unb.br/sigaa/public/docente/portal.jsf?siape=1314342), sendo uma lista que será usada na inspeção da entrega 1, tanto para o grupo, quanto para o grupo+1.
